# DrunkenRecords

Website für das Independent Music Label **DrunkenRecords** — schwarz, cinematisch, neon, voll animiert.

**Live:** https://fgilde.github.io/DrunkenRecords/

## Bands
- **Eyirish** — Irish Punk · Rock · Folk
- **0,5er** — Deutschpop · Rock · HipHop
- **Die PapiBaras** — Kindergerechter Rock

## Dateien
- `index.html` — die fertige, self-contained Seite (wird von GitHub Pages ausgeliefert)
- `DrunkenRecords.dc.html` — bearbeitbares Quellformat (Design Component)
- `image-slot.js` — Komponente für austauschbare Bilder
- `.nojekyll` — verhindert Jekyll-Processing auf GitHub Pages

## Lokal entwickeln / ansehen
Einfach `index.html` im Browser öffnen, oder einen kleinen Server starten:
```
npx serve .
```

## Deployment
GitHub Pages liefert `index.html` aus dem Root des `main`-Branch aus.
